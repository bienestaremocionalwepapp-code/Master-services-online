Referencia rápida del sistema de licencias.
🔄 Flujo del sistema

1. Al iniciar la app cliente → verifica la licencia contra este servidor cada 24h.
2. Si hay conexión → respuesta en tiempo real (ACTIVA / SUSPENDIDA / VENCIDA).
3. Si no hay internet → usa el caché local (válido 72h desde la última verificación).
4. Licencia PERPETUA → una vez activada con la PerpetualKey, no necesita conexión nunca más.
5. Bloqueo → estado BLOQUEADA tiene efecto inmediato (bloqueará en la próxima verificación).

📋 Estados de licencia
ACTIVAFuncionamiento normal
SUSPENDIDABloqueada por falta de pago, mensaje al cliente
VENCIDAExpiró el período + días de gracia
BLOQUEADABloqueo total sin mensaje (fraude, etc.)
PERPETUASin vencimiento, puede funcionar offline
⚙️ URLs de la API
GET ?action=verify&key=SRVT-XXXX... → Verificar licencia (público)
GET ?action=list&token=TU_TOKEN → Listar licencias
GET ?action=suspend&key=X&token=T&reason=Y → Suspender
GET ?action=activate&key=X&token=T → Activar
GET ?action=block&key=X&token=T → Bloquear total
GET ?action=mkperp&key=X&token=T → Convertir a perpetua
